# Sound Generator: Professional Waveform Editor 🎹🔊

A powerful and intuitive **Digital Synthesizer** built with **Flutter**. This utility combines the core principles of sound synthesis with a high-performance **real-time oscilloscope**, allowing users to design and visualize audio waveforms simultaneously.

## 🌟 Key Features
- **Four Core Waveforms:** Instantly switch between Sine, Square, Sawtooth, and Triangle waves.
- **Real-time Oscilloscope:** Visualize sound waves as they are generated, perfect for understanding acoustic physics.
- **Precision Tuning:** Adjust frequencies via intuitive sliders or direct Hz-level numeric input for exact pitch control.
- **LFO Modulation:** Add depth and character with dedicated Frequency and Depth controls for vibrato and effects.
- **Spatial Control:** Integrated stereo panner to manage the audio’s position in the stereo field.
- **Responsive Keyboard:** A piano-style interface optimized for any screen size, providing a comfortable performance experience.

## 💡 Best For...
- **Sound Designers:** Creating unique sound effects (SE) for games or videos.
- **Music Students:** Learning the fundamentals of synthesis and waveform characteristics.
- **Synthesizer Enthusiasts:** A lightweight, high-quality tool for quick inspiration and sonic experimentation.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Low-latency audio buffer generation and custom Painters for real-time waveform visualization (oscilloscope logic).

*Note: This repository contains the core logic and source code. Assets (images, sound fonts, and localization files) are excluded to protect intellectual property.*
